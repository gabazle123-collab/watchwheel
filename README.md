
# WatchWheel 🎬

A stylish Letterboxd-inspired random movie picker.

## Features
- Paste your watchlist
- Random movie generator
- Stylish cinematic UI
- Mobile responsive
- No backend required

## Run locally
Just open `index.html` in your browser.
